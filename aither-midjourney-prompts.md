## Tips for Best Results

- Run each prompt 4 times (default) and pick the best
- Use **Vary (Subtle)** on your favourite to get refined versions
- If too "pretty", add: `gritty, rough, dark, no bloom, no lens flare`
- If not dark enough, add: `deep shadows, underexposed, dark atmosphere, night scene`
- To match UO's classic feel, add: `isometric inspired, retro fantasy RPG, 90s fantasy art`
